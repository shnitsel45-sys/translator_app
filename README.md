Трансформационный русско-английский переводчик
Интеллектуальная система перевода с визуализацией синтаксического анализа и грамматических трансформаций.

Быстрый старт

bash
pip install stanza pymorphy2 deep-translator
python app.py

Основные возможности
Трансформационный перевод с русского на английский с учётом грамматики

Синтаксический анализ (Stanza + Pymorphy2) для определения структуры предложений

Графический интерфейс с вкладками для анализа, правил и перевода

База правил JSON — расширяемая система грамматических шаблонов

Поддержка специальных конструкций: вопросы о цене, владение, фразовые глаголы

Структура проекта

app.py                         # GUI приложение (основной файл для запуска)

main.py                        # Ядро переводчика (синтаксический анализ + трансформации)

transformer_grammar_rules.json # База грамматических правил (SVO, SVOO, вопросы и др.)

Использование
Запустите app.py

Введите русское предложение в поле ввода

Нажмите "Перевести" (или Enter)

Смотрите результат во вкладках:

Перевод — английское предложение

Анализ — структура и зависимости

Правила — применённые грамматические шаблоны

Примеры
"Студент читает книгу" → "Student reads a book"

"Сколько стоит кофе?" → "How much does coffee cost?"

"У меня есть дом" → "I have a house"

Зависимости
Python 3.7+

Tkinter

stanza, pymorphy2, deep-translator

При первом запуске автоматически скачиваются NLP-модели (~5 минут). Требуется интернет для переводов Google.
